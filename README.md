# Практикум по Claude Code для продаж

Интерактивный практикум: 21 урок, где вы под руководством Claude собираете себе рабочие инструменты — CRM, агентов-ресёрчеров, скиллы под холодные письма, коммерческие предложения и разбор звонков.

Уроки запускаются slash-командами: `/L1-setup`, `/L2-onboarding`, `/L3-crm` и так далее.

---

## Как начать с телефона (5 минут)

1. **Нажмите «Use this template» → «Create a new repository»** вверху этой страницы.
   Выберите **Private** — это ваша личная копия, ваши наработки останутся при вас.
2. Откройте **мобильное приложение Claude** → новый чат → подключите **GitHub** и выберите созданный репозиторий.
3. Напишите в чат:
   ```
   /L1-setup
   ```
4. Дальше просто следуйте за Claude. Когда он попросит лицензионный ключ — возьмите его из Телеграм-бота практикума.

Продолжать можно откуда угодно: телефон, браузер, десктоп. Прогресс лежит в репозитории, а не в истории чата — открыли новую сессию, сказали `/L5-security`, поехали дальше.

## Как начать с компьютера

То же самое, только откройте папку репозитория в десктопном Claude Code. Ключ добавляется в файл `.env` через терминал — Claude подскажет точную команду.

---

## Что внутри

| Папка | Что это |
|---|---|
| `.claude/commands/` | Уроки практикума (slash-команды) |
| `CLAUDE.md` | Правила, по которым Claude ведёт вас по курсу |
| `my-project/` | Ваша рабочая папка — профиль, наработки, результаты уроков |
| `aximo-crm/` | Учебная CRM, которую вы соберёте и запустите |
| `aximo-templates/` | Материалы для уроков: транскрипты звонков, договоры, шаблоны КП |
| `aximo-skills/`, `aximo-agents/` | Сюда лягут скиллы и агенты, которых вы напишете |
| `aximo-licensing/` | Проверка ключа и загрузка уроков с курс-сервера |

Тексты уроков со 2-го по 21-й хранятся на сервере и приходят по вашему лицензионному ключу с проверкой подписи — локально их нет.

## Ключ и безопасность

Лицензионный ключ живёт в файле `.env`, который **исключён из git** (см. `.gitignore`) — он не уедет в репозиторий даже случайно. Ключ приходит в Телеграм-боте в момент оплаты.

## Что-то не работает

Напишите в Телеграм-бот практикума — тот же чат, где пришёл ключ.

# Парсинг портала участников конференции

## Шаг 0 — выбрать способ доступа

Портал закрыт логином. Есть три способа зайти — выбирай по тому, что доступно прямо сейчас:

- **(а) Браузер-коннектор** — если доступно управление браузером (есть инструменты браузер-коннектора): работаем по уже открытой залогиненной вкладке. Логин делает человек сам, креды в чат не просим. Единственный путь туда, где вход только под личным аккаунтом — Sales Navigator, корпоративные кабинеты, SSO.
- **(б) Скрипт с авторизацией** — если на руках есть логин/пароль (Basic Auth), API-ключ или токен: тянем страницы напрямую HTTP-запросом и парсим разметку. Быстро, повторяемо, ставится на расписание. Креды берём из `.env` или из задачи; в вывод и в чат их не печатаем.
- **(в) Сниппет для консоли браузера** — если нет ни (а), ни (б): даём короткий JS (`document.querySelectorAll` по карточкам → JSON/CSV), человек выполняет его в DevTools и присылает вывод.

Если Claude работает в облачной сессии, браузера пользователя у него нет — способ (а) недоступен, по умолчанию используем (б), скрипт с авторизацией.

## Шаг 1 — пройти весь список

Прочитать каталог участников. Если есть пагинация / «показать ещё» / бесконечная прокрутка — пролистать/подгрузить до конца (в скриптовой ветке — перебрать страницы по параметру), собрать всех, а не первый экран.

## Шаг 2 — вытащить поля

По каждому участнику: имя, должность, компания, сайт (+ LinkedIn/любые публичные поля, если на карточке есть). Чего нет — оставить пустым, не выдумывать.

## Шаг 3 — собрать CSV

Дедуп по имени+компании. Записать `aximo-skills/conf-parser/participants.csv` (колонки: `name, title, company, website, source_url`). Показать табличку в чате.

## Формат вывода

Путь к `participants.csv` (кликабельно) + таблица (Имя · Должность · Компания · Сайт) + строка «всего N участников» + одна строка, каким способом забрали данные.

## Гейты безопасности

- Только читаем портал и пишем CSV — ничего на портале не меняем и не отправляем.
- Поля не выдумываем — чего нет, оставляем пустым.
- Пароли/ключи в чат и в лог не выводим.
- Письма участникам — отдельным шагом, только черновики (человек в цикле).

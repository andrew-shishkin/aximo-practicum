# Твои навыки (Skills)

Здесь живут **Skills**, которые ты создаёшь по ходу практикума — твой растущий тулкит.

Пока пусто 🙂 Первый навык появится в **Уроке 4** («Утренний брифинг и приоритеты на день»). Дальше папка будет наполняться: чистка CRM, поиск контактов, фолоапы и другое.

Каждый навык — это папка `<имя-навыка>/SKILL.md` (инструкции, по которым работает Claude). К каждому есть **кнопка** — slash-команда вида `/имя-навыка` для быстрого запуска.

На финале курса мы «выпустим» твои навыки в глобальное окружение (`~/.claude/`), чтобы команды работали всегда — в любом проекте и уже по твоей реальной CRM.

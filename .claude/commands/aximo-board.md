Покажи ученику доску Aximo CRM.

1. **Пересобери доску из данных** (чтобы она отражала актуальное состояние CRM). Сборщик есть в двух видах — результат байт-в-байт одинаковый, бери тот рантайм, что доступен:
   - **`python3 aximo-crm/_build.py`** (на Windows — `python aximo-crm/_build.py`) — основной путь.
   - либо **`node aximo-crm/_build.js`** — если node установлен и так удобнее.
   Если ни `python3`, ни `node` нет или команда упала — не страшно, `aximo-crm/board.html` уже самодостаточен.

2. **Дальше — по среде ученика** (поле **среда** в `my-project/student.md`, правило 21).

   **Если `local`** — открой доску командой под ОС:
   - macOS: `open aximo-crm/board.html`
   - Windows: `start aximo-crm/board.html`
   - Linux: `xdg-open aximo-crm/board.html`

   **Если `cloud`** — **ОТПРАВЬ ученику файл `aximo-crm/board.html` в чат.** Именно отправь как файл, а не дай ссылку и не опиши словами: тап по присланному файлу открывает настоящую доску — канбан, свайп по колонкам, раскрывающиеся карточки. Это лучшее, что есть, и ради этого всё и делалось.

   Инструмент отправки файла в этой команде доступен (ограничений на инструменты у неё нет). Если в твоём окружении он называется иначе — используй тот, что реально прикладывает файл к сообщению.

   **Проверь, что файл действительно ушёл, и только потом говори про него.** Фраза «тапни по файлу выше» без фактически отправленного файла — обман: тапать будет некуда. Это уже случалось.

   **Если отправить файл не получилось** — не выкручивайся и не молчи. Есть готовый запасной вариант: рядом лежит **`aximo-crm/board.md`** — та же доска обычным текстом (~5 КБ). Прочитай его инструментом Read и покажи содержимое как есть. Выглядит скромнее, но ученик увидит всю картину и урок не встанет.

3. **Ссылку на `board.html` давай ТОЛЬКО в среде `local`** — там она кликабельна и открывает доску: [Aximo CRM](aximo-crm/board.html). **В `cloud` ссылку не давай вообще** — она не кликается и только раздражает.

4. **Только для `local`:** коротко назови состояние словами — сколько сделок по стадиям и на какую сумму открытый пайплайн. В `cloud` этого не нужно: ученик получил доску целиком.

5. Скажи ученику примерно так:
   - **`cloud`:** «Держи актуальную доску — тапни по файлу, откроется. Сверху три вкладки: **Сделки** (канбан по стадиям), **Компании** и **Контакты**. Нужно свежее состояние — просто напиши `/aximo-board`.»
   - **`local`:** «Доска актуальна, открывай по ссылке. Сверху три вкладки: **Сделки**, **Компании** и **Контакты** (клик по заголовку сортирует). Нужно свежее состояние — напиши `/aximo-board`.»

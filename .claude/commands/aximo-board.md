Покажи ученику доску Aximo CRM.

1. **Пересобери доску из данных** (чтобы она отражала актуальное состояние CRM). Сборщик есть в двух видах — результат байт-в-байт одинаковый, бери тот рантайм, что доступен:
   - **`python3 aximo-crm/_build.py`** (на Windows — `python aximo-crm/_build.py`) — основной путь.
   - либо **`node aximo-crm/_build.js`** — если node установлен и так удобнее.
   Если ни `python3`, ни `node` нет или команда упала — не страшно, `aximo-crm/board.html` уже самодостаточен.

2. **Дальше — по среде ученика** (поле **среда** в `my-project/student.md`, правило 21).

   **Если `local`** — открой доску командой под ОС:
   - macOS: `open aximo-crm/board.html`
   - Windows: `start aximo-crm/board.html`
   - Linux: `xdg-open aximo-crm/board.html`

   **Если `cloud`** — командами ничего не открывай (открывать некуда, это песочница). И **не рассчитывай на markdown-ссылку**: тап по ней ничего не делает. Работает только карточка файла, которую приложение показывает после того, как ты этот файл **реально открыл инструментом Read**.

   **ОБЯЗАТЕЛЬНО ВЫЗОВИ Read на `aximo-crm/board.html`. Без этого вызова карточки не будет, и фраза «тапни по файлу выше» окажется враньём — ученику будет некуда тапать.**
   Файл большой (~107 КБ), поэтому читай его **частично**: Read с параметром `limit` на 5–10 строк. Этого достаточно, чтобы файл появился в интерфейсе, и контекст не забивается. Прочитанное в чат не пересказывай и не показывай — сам вызов и есть цель.

   Только ПОСЛЕ фактического вызова Read скажи: «доска пересобрана — тапни по файлу `board.html` выше». Если Read почему-то не отработал, **не произноси эту фразу**: скажи честно, что доску открыть не получилось, и опиши состояние словами (пункт 4).

3. **Ссылку на доску давай всегда**, в любой среде: [Aximo CRM](aximo-crm/board.html). В `local` она кликабельна и открывает файл; в `cloud` это просто указание пути, а открывается доска тапом по карточке файла из пункта 2.

4. **Коротко назови текущее состояние словами** — прочитай `aximo-crm/data.js` и скажи в одну-две строки: сколько сделок по стадиям и на какую сумму открытый пайплайн. Это важно: так ученик понимает картину, даже если доску не открывал. JSON не показывай.

5. Скажи ученику примерно так:
   «Доска актуальна. Сверху три вкладки: **Сделки** (канбан по стадиям), **Компании** и **Контакты** (таблицы, клик по заголовку сортирует). Нужно свежее состояние — просто напиши `/aximo-board`, пересоберу.»

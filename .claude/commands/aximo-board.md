Покажи ученику доску Aximo CRM.

1. **Пересобери доску из данных** (чтобы она отражала актуальное состояние CRM). Сборщик есть в двух видах — результат байт-в-байт одинаковый, бери тот рантайм, что доступен:
   - **`python3 aximo-crm/_build.py`** (на Windows — `python aximo-crm/_build.py`) — основной путь.
   - либо **`node aximo-crm/_build.js`** — если node установлен и так удобнее.
   Если ни `python3`, ни `node` нет или команда упала — не страшно, `aximo-crm/board.html` уже самодостаточен.

2. **Дальше — по среде ученика** (поле **среда** в `my-project/student.md`, правило 21).

   **Если `local`** — открой доску командой под ОС:
   - macOS: `open aximo-crm/board.html`
   - Windows: `start aximo-crm/board.html`
   - Linux: `xdg-open aximo-crm/board.html`

   **Если `cloud`** — **показывай доску прямо в чате.** Не давай ссылок, не проси никуда тапать, не обещай карточек файлов: на телефоне это не работает, проверено трижды.
   Сборка создаёт рядом файл **`aximo-crm/board.md`** — ту же доску обычным текстом (~5 КБ). **Прочитай его инструментом Read и покажи ученику содержимое как есть** — это и есть доска. Она сразу видна в переписке, открывать ничего не нужно.
   Ничего к ней не дописывай и не пересказывай своими словами: markdown отрисуется сам, заголовки стадий и суммы уже внутри.

3. **Ссылку на `board.html` давай ТОЛЬКО в среде `local`** — там она кликабельна и открывает доску: [Aximo CRM](aximo-crm/board.html). **В `cloud` ссылку не давай вообще** — она не кликается и только раздражает.

4. **Только для `local`:** коротко назови состояние словами — сколько сделок по стадиям и на какую сумму открытый пайплайн. В `cloud` этого не нужно: ученик видит всю доску целиком прямо в чате.

5. Скажи ученику примерно так:
   - **`cloud`:** «Вот актуальная доска. Нужно свежее состояние — просто напиши `/aximo-board`.»
   - **`local`:** «Доска актуальна, открывай по ссылке. Сверху три вкладки: **Сделки** (канбан по стадиям), **Компании** и **Контакты** (таблицы, клик по заголовку сортирует). Нужно свежее состояние — напиши `/aximo-board`.»
